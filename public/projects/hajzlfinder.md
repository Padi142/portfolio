## TLDR
Silly idea I had about a website that tracks public restrooms, their prices and if you need a special code to access it like in a restaurant. It sat with almost not traffic for a while, but then all of a sudden it got quite popular with up to 50k visitors a day. Now it is sitting comfortably at 2-3k visitors a week. The website is built using [NextJS](https://nextjs.org/) and [Convex](https://convex.dev/) for real time synced data between all clients.
